# CMD 命令
## 文件
- 新建空文件 `type nul>.` 
```
type nul> newtest.txt 
```
- 新建非空文件 `echo [fileContent]>.`
```
echo [fileContent]>newtest.txt
```
- 删除文件 `del*.*`
```
del newtest.txt
```
## 文件夹
文件夹
### 新建文件夹 `md/mkdir  folderName`
- 【盘符：\】【路径\】
```
md c:\test\newtest
md newtest 
mkdir newtest
```
### 删除文件夹 `rd folderName`
