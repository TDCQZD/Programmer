# 设计模式

## 为什么学习设计模式？
* 了解在软件开发过程中所面临的问题的最佳解决方案。
* 优化软件系统设计，少写代码，少出Bug。
* 增加软件系统的可复用性和扩展性，少写代码。

## [设计模式简介](./设计模式简介.md)

## 如何学习？
1. 了解设计模式，掌握设计模式六大原则
2. 单个设计模式：
    * 理解该模式的理论基础，掌握应用场景、优缺点、注意事项
    * 设计应用实例
        - 实例UML图
        - 实例代码（Java/Go）
